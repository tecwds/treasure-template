## Treasure template

此模板是为了快速**部署**或**开发**一个使用 `springboot` 和 `nodejs` 的前后端项目。


## 功能特点
 
**1. 开发**

- [ ] 使用 `vscode` 直接开发
- [ ] 使用 `jetbrains` 直接开发
- [ ] 在 `vscode` 使用 `devcontainer` 开发
- [ ] 在 `jetbrains` 使用 `devcontainer` 开发

**2. 中间件**

- [ ] `mysql` 容器
- [ ] `mongodb` 容器
- [ ] `redis` 容器
- [ ] `nginx` 容器
- [ ] `elk` 容器

**3. 构建**

- [ ] 本地构建
- [ ] 远程构建
- [ ] `Github Actions` 构建
- [ ] 使用 `docker-compose` 构建

**4. 部署**

- [ ] 本地部署
- [ ] 远程部署
- [ ] 使用 `Dockerfile` 部署
- [ ] `Github Actions` 部署
- [ ] 使用 `docker-compose` 部署

## 目录说明

| 目录 | 说明 |
| ----| ----|
| `.devcontainer` | 开发容器配置 |
| `.github` | github 相关文件 |
| `docs` | 文档 |
| `src` | 源码位置 |
| `scripts` | 脚本 |
| `env` | 环境变量 |

## 开始使用

1. fork 本项目, 或者 clone 本项目, 亦或者使用此仓库作为模板创建一个新的仓库
2. 将代码复制到 src 目录下（或者新创建一个）
