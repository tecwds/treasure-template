## Treasure template

写 `springboot` 或者 `nodejs` 项目的时候，往往环境难以搭建（或者难以部署），因此我创建了一个模板，用于解决这个问题。

## Features

1. `devcontainer`
2. `vscode support`
3. `jetbrains support`


## 目录说明

| 目录 | 说明 |
| ----| ----|
| `.devcontainer` | 开发容器配置 |
| `.github` | github 相关文件 |
| `docs` | 文档 |
| `src` | 源码位置 |
| `scripts` | 脚本 |
| `env` | 环境变量 |

## 使用

1. fork 项目，或者 clone 项目，或者作为模板创建一个新的仓库
2. 修改 `env` 下的配置文件
